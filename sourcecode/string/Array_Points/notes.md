# 指向字符串的指针

* 将指针指向字符串
  * 可以指向常量字符串
  * 也可以指向存储字符串的字符数组


    `Input: `

    ```C
    char * words = "My heart is still. ";
    words += 9;
    puts(words);
    ```

    |M|y| |h|e|a|r|t| |i|s| |s|t|i|l|l|.|\0|
    |-|-|-|-|-|-|-|-|-|-|-|-|-|-|-|-|-|-|-|
    
    M <-- words

    |i|s| |s|t|i|l|l|.|\0|
    |-|-|-|-|-|-|-|-|-|-|
    
    i <-- words

    `Output: `

    ```
    is still.
    ```
    
  * 数组形式和执行形式的不同
    * 初始化字符数组时会把静态存储区的字符串拷贝到数组中
    * 初始化指针时只把字符串的地址拷贝给指针
