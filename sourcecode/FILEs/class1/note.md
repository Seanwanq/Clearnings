# 文件基本操作

1. 如何打开文件
   1. 路径
      1. 相对路径
      2. 绝对路径(Path)
   2. 打开方式
      * r: 只读 read 不带有创建文件的功能!
      * w: 只写 write 如果文件不存在, 则创建该文件;如果存在，则清空该文件
      * a: 追加模式 append 在原文件的末尾接着写
      * +: 可读可写
      * b: 二进制 binary
      * r+, w+, a+
      * rb+, wb+, ab+
      * rb, wb, ab
2. 如何关闭文件
   1. fclose(文件指针);
